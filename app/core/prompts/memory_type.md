偏好:喜欢、不喜欢、习惯、最喜欢的媒体
健康:身体健康、心理健康、饮食、睡眠

工作:工作角色、公司、项目、晋升
·教育:课程、学位、认证、技能发展
项目:待办事项、里程碑、截止日期、状态更新
AI、ML与技术:基础设施、算法、工具、研究
技术支持:错误报告、错误日志、修复
财务:收入、支出、投资、账单
购物:购买、愿望清单、退货、交付
法律:合同、政策、法规、隐私
娱乐:电影、音乐、游戏、书籍、事件

客户支持:票务、查询、解决方案
.产品反馈:评分、错误报告、功能请求
新闻:文章、头条、趋势话题
组织:会议、约会、日历
.目标:抱负、KPI、长期目标
指南:
1、仅返回JSON格式中'categories'键下的类别。
